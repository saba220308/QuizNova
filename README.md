
# 🧠 QuizNova – Interactive Quiz Application

An interactive and beginner-friendly web-based Quiz Application that allows users to attempt multiple-choice questions, get instant results, and improve their knowledge in a simple and engaging way.

---

## 📌 Project Overview

**QuizNova** is a lightweight quiz platform designed for students and learners to practice questions and evaluate their knowledge instantly. The application provides a clean UI and smooth user experience.

---

## ❗ Problem Statement

Many learners struggle to find a simple and distraction-free platform for practicing MCQ-based questions. Existing platforms are often complex or overloaded with features. This project solves that by providing a minimal, fast, and interactive quiz system.

---

## 🎯 Objectives

- Provide an easy-to-use quiz platform
- Display MCQ-based questions clearly
- Evaluate answers instantly
- Show final score at the end of the quiz
- Improve self-learning and practice

---

## 🚀 Features

- 📝 Multiple Choice Questions (MCQs)
- ⚡ Instant score calculation
- 🔄 Restart quiz option
- 🎯 Simple and clean UI
- 📱 Responsive design (mobile + desktop)
- ⏱️ Optional timer per question

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend (Optional):** Python Flask / Node.js  
- **Data Storage:** JSON / Database (optional)
